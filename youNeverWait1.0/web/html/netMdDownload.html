Once you download please follow the below steps

  • Unzip netmdrt.rar to your system.
  • Set Environmet variable for that path as NETMDRT.

  • Install Postgresql using "postgresql-9.1.4-1-windows" installer from "\netmdrt\postgresql\pgsql\bin".
  • Install tomcat using "apache-tomcat-6.0.35" installer from "\netmdrt\tomcat\bin".
  • Put the downloaded war file to "<Tomcat installation path>\Tomcat 6.0\webapps\".
  • Put the attached context.xml to "\Tomcat 6.0\conf".
  • Download the attached postgresql.jar files(4 files) to "\Tomcat 6.0\lib".
  • Download and place the attached netmd.properties file to "\Tomcat 6.0\conf".
  • Create a database named netmd in PgAdmin and restore the database with attached backup file.
  • To access the application Enter the URL " http://localhost:8080/netmd/ws/ui/html/startUp/" to your browser.

    • Notes

  • To set Environment Variable go to "My Computer/Properties/Advanced/Environment variables/System Variables" and make a new variable,Variable name : NETMD Variable value : Tomcat installation path \Tomcat 6.0.
  • While installing postgresql, in the last window of installation please untick stack builder option.
  • Go to PostgresSQL installation path \PostgreSQL\9.1\bin, Click on the pgAdmin->Right click on the Databases. Select NewDatabase Name : netmd, Owner : Postgres Press'ok'button . Right click on the database (netmd) Give details as Format : Custom and tar, File name :browse the attached backup file , Role name : Postgres Press "Restore" button Ok done

System Requirements

Windows 2003 or later
