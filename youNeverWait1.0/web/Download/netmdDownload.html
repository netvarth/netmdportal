Once you download please follow the below steps

  • Unzip netmdrt.rar to your system.
  • Set Environmet variable for that path as NETMDRT.

  • Install Postgresql using "postgresql-9.1.4-1-windows" installer from "\netmdrt\postgresql\pgsql\bin".
  • Install tomcat using "apache-tomcat-6.0.35" installer from "\netmdrt\tomcat\bin".
  • Put the downloaded war file to "<Tomcat installation path>\Tomcat 6.0\webapps\".
  • Copy the file context.xml from "\netmdrt\tomcat\conf" and replace it with context.xml in "<Tomcat installation path>\Tomcat 6.0\conf".
  • Copy postgresql.jar files(4 files) from "\netmdrt\tomcat\lib" to "<Tomcat installation path>\Tomcat 6.0\lib".
  • Run createDBscript.batfrom “\netmdrt\bin”.

  • Enter the URL " http://localhost:8181/netmd/ws/ui/html/startUp#" to your browser.
  • Login Using, Username= admin and Password= netvarth.
  • Restart tomcat server. (Go to "Control panel/Administrative tools/Services/Apache Tomcat 6").
  • Copy the downloaded war to C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps.
  • Enter the URL " http://localhost:8181/netmd/ws/ui/html/startUp#" to your browser and Login to NetMd.

    • Notes

  • To set Environment Variable go to "My Computer/Properties/Advanced/Environment variables/System Variables" and make a new variable.
  • While installing postgresql, in the last window of installation please untick stack builder option.
  • The default port number on which tomcat being installed is 8080. If you insatall it on another port, you will have to change in it the URL.
  • Replace the latest createdb.sql and insert_tables.sql in "\netmdrt\bin" with new one when issued
  • We can see the log files in "<Tomcat installation path>\Tomcat 6.0\logs".

System Requirements

Windows 2003 or later
